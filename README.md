KildeviserSearchSDK
===================

Installation
------------

Get dependencies with `npm install`

Building
--------

    gulp build

or

    guilp buildDev


Development
-----------

To start a development server, run

    gulp watch

Profiles
--------
Set different profiles using the --profile flag while building.
Set profile preferences in the js-files in /profiles
